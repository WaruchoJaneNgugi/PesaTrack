# ExpenseFlow - Expense Tracking System

A full-featured expense management application with employee and admin roles, built with React TypeScript and Firebase.

## Features

- **Authentication**: Phone number + 4-digit PIN login/signup with role selection.
- **Employee Dashboard**: Create expense requests, view status, upload receipts after approval.
- **Admin Dashboard**: Approve/reject requests, view all requests, view uploaded receipts.
- **Audit Logs**: All actions logged for transparency.

## Prerequisites

- Node.js 16+
- Firebase account (free tier works)
